# webapp-main HOWTO
## Introduction
```
webapp is based on yeoman, contained YO, Grunt and bower.
web component: angularJS, bootstrap, angular-UI
```
## Quick Start
* Install environment
firstly, NODEJS runnable environment is needed.
```
gem install compass
npm install -g yo
cd webapp-main
npm install
bower install --save
```
* Start development service 
```
grunt serve
```
* Generate dist
```
grunt build
```
